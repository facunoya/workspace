Ahora vamos a ver la propiedad de display!

Esta es una propiedad inline siginifica que sigue escribiendo a continuacion, no abajo(span viene con display inline)
div, p, todos los h, header, footer, section son tipo bloque"block"
span, link e imagenes son inline
si ponemos display: none los elementos se vuelven invisibles.

con la propiedad de display le podemos cambiar el display que traen por defecto.
